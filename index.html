Import Tax Sale Data for Any MD County

CSV Format

Create a CSV file with these columns (headers required):

year,parcel,owner_name,assessment,bid_amount,sale_amount,bid_premium,total_due,bidder_name,county_code

county_code is the 2-digit MD SDAT county code. The tool uses it to build the API account ID:
account_id = county_code + "01" + parcel.substring(1)

Tips

• The parcel is the 12-digit number from the tax sale PDF
• Different counties may have different PDF layouts — you will need to extract the data into CSV format
• The district code defaults to "01" — if your county uses a different district, add a district column
• Assessment from the CSV is the PDF/stale value — the tool fetches the LIVE current value automatically
